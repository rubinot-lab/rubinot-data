# rubinot-data Endpoint Planning

This folder maps current Rubinot endpoint exposure (from `rubinot-live`) to a TibiaData-inspired target contract for `rubinot-data`.

## Intent

- Preserve existing consumer value
- Normalize naming and contracts
- Align behavior with TibiaData maturity patterns (versioning, restrictions, validation-first)

## Files

- `current-rubinot-live-endpoints.md` — current endpoint URLs as used today
- `target-endpoints-v1.md` — proposed `rubinot-data` endpoint contract
- `compatibility-mapping.md` — mapping from current routes to target routes
